Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>. (Não semântica).

Nessa frase temos um termo em negrito usando a tag <strong>. (Semântica).

Visualmente, não tem diferença entre as duas. No entanto, a formatação com <b> só tem uma função de FORMA, que devia ser feita com o CSS. Já o <strong>, além da forma, dá um SENTIDO para o jeito que aquilo está formatado.

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i>. (Não semântica).

Nessa frase, temos um termo em ênfase> usando a tag <em> (Semântica).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Tag <ins>.

Existe também o sublinhado com a tag <u>. (Não semântica).
